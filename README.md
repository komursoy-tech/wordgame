# Kelime Oyunu - Mobil Uygulama

Bu proje, React Native ve Expo kullanılarak geliştirilmiş, kullanıcıların kelime bilgisini test eden interaktif bir mobil uygulamadır.

## 🚀 Özellikler
* **Dinamik Soru Havuzu:** Farklı kategorilerde rastgele soru seçimi.
* **Zaman Yönetimi:** Kullanıcının hızını ölçen geri sayım sayacı.
* **Skor Sistemi:** Doğru cevaplarla puan kazanma mekanizması.
* **Modern Arayüz:** Kullanıcı dostu ve performans odaklı UI tasarımı.

## 🛠 Kullanılan Teknolojiler
* **Framework:** React Native
* **Araç Zinciri:** Expo
* **Dil:** JavaScript / TypeScript


## ⚙️ Kurulum
Projeyi yerel cihazınızda çalıştırmak için:
1. Depoyu klonlayın: `git clone https://github.com/komursoy-tech/wordgame.git`
2. Bağımlılıkları yükleyin: `npm install`
3. Uygulamayı başlatın: `npx expo start`
